# KeenKeeper – Friendship Tracker

KeenKeeper is a beautiful web application that helps you track and nurture your meaningful friendships. It allows you to log calls, texts, and video chats with friends, view a timeline of interactions, and get analytics on your friendship patterns.

## 🚀 Technologies Used

- **React 18** – UI library
- **React Router DOM** – Navigation
- **Tailwind CSS** – Styling & responsiveness
- **Recharts** – Pie chart for analytics
- **React Hot Toast** – Notifications
- **Vite** – Build tool

## ✨ Key Features

1. **Friend Dashboard** – View all friends in a responsive grid with status badges (overdue, almost due, on-track). Each card shows profile picture, days since contact, and tags.

2. **Interaction Timeline** – From any friend's detail page, log a Call, Text, or Video. Each interaction is saved and appears on the global Timeline page with filtering by type.

3. **Friendship Analytics** – Visual pie chart showing distribution of calls, texts, and videos. Summary statistics are also displayed.

4. **Responsive Design** – Works perfectly on mobile, tablet, and desktop.

5. **404 Page** & Loading Animation – Graceful error handling.
